# 🤖 AIPOS

> **The operating system for AI-powered software development.**

AIPOS is an open-source framework that helps AI coding agents build production-ready software through structured documentation, persistent project memory, standardized workflows, and engineering best practices.

## ✨ Why AIPOS?

AI coding agents are powerful, but they often:

- Forget previous decisions
- Break project architecture
- Duplicate code
- Ignore coding conventions
- Lose project context

AIPOS solves these problems by providing a consistent framework that any AI agent can follow.

## 🚀 Features

- 🧠 Persistent Project Memory
- 📋 Structured Milestones
- 🏛 Architecture Guidelines
- 💻 Coding Standards
- 🔄 Development Workflow
- 📊 Project Status Tracking
- 📜 Decision Log
- 🤖 AI Agent Playbook

## 🤖 Supported AI Agents

AIPOS is AI-agnostic and works with any coding agent, including:

- Claude Code
- OpenAI Codex
- Gemini CLI
- Cursor
- Cline
- Roo Code
- Continue
- Windsurf
- And future AI coding agents
